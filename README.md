# Quiz Cidadania Portuguesa

Quiz interativo para ajudar visitantes a identificar direito à cidadania portuguesa por descendência, casamento ou residência.

## Funcionalidades planejadas
- Perguntas sobre pais, avós, bisavós
- Ramificação para bisnetos: avô/avó vivo?
- Resultados personalizados
- Disclaimer legal

## Estrutura
- index.html com JS para quiz
- CSS moderno
- Mobile first

## Atualizações 2026
- Requisitos atualizados conforme Lei Orgânica n.º 1/2026
- Foco em encadeamento de processos quando ascendentes vivos.
